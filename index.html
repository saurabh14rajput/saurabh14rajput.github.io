Latest Projects

User level thread library

Latest

Designed, developed, and implemented a user level thread library (like Pthread) in C.

Implemented multilevel priority queue scheduler for the library. Resolved priority inversion issue by implementing priority inheritance. Another major problem observed with multilevel priority queue is starvation. This issue was resolved with implementing thread aging. There is also sometimes the possibility of Smart voluntary yield attack by a program if it is able to figure out the queue quanta. This possibility was nullified by restricting the total yield time for each thread at each priority level.
